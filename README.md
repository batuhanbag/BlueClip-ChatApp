# BlueClip Chat App

This project is a chat application prepared for BlueClip company using react native w/typescript and firebase infrastructure.

## Before the installation

Before the installation please don't forget the put firebase config values in firebase.ts file (app/config/firebase.ts).

## Installation

Use yarn for install dependencies

```bash
yarn
```

## Test project on your local - IOS

```javascript
yarn ios
// OR
expo start --ios
```

## Test project on your local - ANDROID

```javascript
yarn android
// OR
expo start --android
```
